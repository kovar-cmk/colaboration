// Profile

About

I build and break network infrastructure for a living. My work sits at the intersection of Cisco datacenter architecture and offensive/defensive security — from designing ACI fabrics and automating NX-OS deployments to finding the gaps that others miss.

I don't do checklist security. I think in kill chains, attack graphs, and defense-in-depth layers. Every network I touch gets stress-tested from both sides: what would I exploit? and what would I defend?

Background in enterprise networking — Nexus 9K, Catalyst, Firepower, DNA Center. Transitioned into security because I kept finding the same misconfigurations, the same blind spots, the same "it works in production" excuses. Now I fix them before someone else exploits them.
